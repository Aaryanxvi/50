# Yemee's 50th Birthday - Goa 2026

A high-end, artistic landing page for Yemee's 50th Birthday celebration in Goa.

## Project Overview
This project is a modern React application built with:
- **Vite**: For fast development and building.
- **Tailwind CSS (v4)**: For utility-first styling with a custom high-end configuration.
- **Framer Motion**: For smooth, premium animations.
- **React**: For component-based architecture.

## Features
- **Artistic "Editorial" Design**: Massive typography and asymmetrical layouts.
- **RSVP Form**: Dynamic form for guest attendance.
- **Countdown Timer**: Custom timer for the event date.
- **Responsive Layout**: Optimized for all devices.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```
